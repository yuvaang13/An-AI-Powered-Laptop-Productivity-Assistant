## Key Differences: Swift → Electron

| Feature | Swift | Electron |
|---------|-------|----------|
| UI | SwiftUI | React + Framer Motion |
| Window | NSPanel | BrowserWindow |
| Transparency | NSVisualEffectView | backdrop-filter CSS |
| App Detection | NSWorkspace | AppleScript/PowerShell |
| Browser URL | AppleScript | AppleScript/PowerShell |
| Permissions | AXIsProcessTrusted | Manual permission flow |
| Storage | UserDefaults | electron-store (JSON) |
| Menu Bar | NSStatusBar | Tray API |

## Risk Assessment

### High Priority
1. **Accessibility permissions** - May require manual user intervention on both platforms
2. **Browser detection** - Different APIs per platform and browser
3. **Window positioning** - Different coordinate systems (macOS has flipped Y-axis)

### Medium Priority
1. **Performance** - Polling overhead vs native event listeners
2. **Build signing** - Code signing required for macOS distribution
3. **Overlay click-through** - Proper implementation varies by platform

### Low Priority
1. **Animation fidelity** - Framer Motion may have subtle differences
2. **System theme detection** - May need manual light/dark mode handling
